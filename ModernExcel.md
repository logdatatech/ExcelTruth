
# What Is Modern Excel

> "Modern Excel" is a phrase used to describe the suite of new tools.

* Get & Transform (Power Query)
* M is for DATA Monkey the Language of Power Query
* Power Pivot
* Power View
* Power BI (can share and round trip in Excel)
